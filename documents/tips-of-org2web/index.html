1 How to install org2web in manual way

To install org2web manually you should first install all dependencies listed in README.org. Once this has been completed, clone the repo: 

git clone https://github.com/tumashu/org2web.git

After that, please remember to add this location to your emacs' load-path variable so that emacs can find the package. 

(add-to-list 'load-path "/path/to/org2web")
(require 'org2web)

NOTE: This is NOT recommended way for emacs beginner.

2 How to install org2web Through package management system

  1. Setting melpa repository, see: http://melpa.org/#/getting-started

  2. Run the following command 

    M-x package-install RET org-website RET

  3. Add the following to your ~/.emacs file: 

    (require 'org2web)

3 How to quickly add a new post

M-x org2web-new-post

This command will ask you the follow question:

  1. Which project do you want post?
  2. Category?
  3. Filename?

4 How to quickly insert org-website post template

M-x org2web-insert-options-template

5 How to configure the default slogan

Add the follow two lines to you config alist. 

:site-main-title "your main slogan"
:site-sub-title "your sub slogan"

6 How to add an avatar to the page?

Add the follow line to you config alist. 

:personal-avatar "URL to an image"

Image URL example:

  1. http:/XXXXX.com/XXXX.jpg
  2. /media/img/XXXXX.jpg
  3. /assets/XXX/XXX.jpg

7 How to set html git remote?

Add the follow line to you config alist. 

:remote (git "<your-html-git-remote-url>" "<branch-name>")

8 How to add a github link

Add the follow line to you config alist. 

:source-browse-url ("GitHub" "https://github.io/<username>/<project-name>")

9 How to do site traffic analytics with Google Analytics?

Add the follow line to you config alist. 

:personal-google-analytics-id "your google analytics id"

10 How to disable commenting for posts under certain categories?

(setq org2web-category-config-alist
      (cons '("photography" ;; category name goes here
              :show-comment nil)
            org2web-category-config-alist))

Other config items:

  1. :show-meta: show post meta info at the bottom of post?
  2. :uri-generator: the function used to generate uri for posts under that category (however, it is not recommended to customize except you are an expert)
  3. :uri-template: the template used to generate uri
  4. :sort-by: how to sort posts on category index page, by
  5. :date or by :mod-date (:mod-date is last modification date)?
  6. :category-index: generate an index page for this category?
